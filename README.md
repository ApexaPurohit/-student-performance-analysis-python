# -student-performance-analysis-python
Student Performance Analysis using Python - Pandas, NumPy, Matplotlib, Seaborn


## Tools
Python | Pandas | NumPy | Matplotlib | Seaborn

## Domain
Education

## Dataset
1,000 students | 8 columns | Source: Kaggle

## About
Analyzed key factors affecting student exam scores
using Python-based data analysis and visualization.

## Key Analysis
- Impact of lunch type on student performance
- Effect of test preparation course on scores
- Gender-wise performance comparison
- Parental education influence on exam results

## Key Findings
- Standard lunch students score ~11 points higher in math
- Test preparation course improves scores by 5-8 points
- Male students perform better in math, females lead in reading and writing
- Higher parental education linked to better student performance
